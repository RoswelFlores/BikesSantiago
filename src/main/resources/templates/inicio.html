Usuario

Si eres usuario sin credenciales de acceso solo podras ver el listado de clientes, para ver las facturas que tiene el cliente debes contar con credenciales de empleado o administrador.

Empleado

Con las credenciales de empleado de la empresa Bikes Santiago, podras ver el listado de clientes y tambien la cantidad de facturas que esten relacionadas con cada cliente.

Administrador

Con las credenciales de administrador de la empresa Bikes Santiago, podras ver el listado de clientes y tambien la cantidad de facturas que esten relacionadas con cada cliente ademas de poder agregar, eliminar o editar clientes en la base de datos. Como administrador contaras con la posibilidad de agregar, editar o eliminar las facturas de cada cliente y tambien el detalle de cada una de ellas.
